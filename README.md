Local development
-----------------

Use [Jekyll](http://jekyllrb.com):

~~~ sh
$ gem install jekyll
$ jekyll serve -P 4567
~~~

Then hack away:

~~~ sh
$ open http://localhost:4567/
$ vim index.md
~~~
